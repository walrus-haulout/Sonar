# SONAR Protocol Roadmap

**Building the future of decentralized audio datasets, from hackathon MVP to global creator network**

---

## Overview

SONAR Protocol is developing a comprehensive ecosystem for decentralized audio dataset storage, verification, and marketplace. Our roadmap is structured in three phases, each building upon the previous to create a robust, scalable, and community-driven platform.

---

## Phase 1: NOW (Hackathon MVP) âœ…

**Status:** Completed
**Timeline:** Current
**Focus:** Core infrastructure for decentralized audio storage, encryption, and marketplace

### Key Deliverables

#### Walrus Storage Integration
- Decentralized file hosting on Walrus Network
- Blob storage with content addressing
- Redundant storage across multiple nodes
- CDN-like delivery for audio streaming

#### SEAL Encryption
- End-to-end encryption using Mysten SEAL
- Access control via blockchain policies
- Encrypted data remains private until purchase
- Key distribution through on-chain verification

#### Audio Verification System
- Pre-encryption quality checks
- Audio format validation (MP3, WAV, FLAC)
- Duration and bitrate verification
- Metadata extraction and validation

#### Spam Prevention
- **1 SUI upload fee** to deter spam submissions
- Economic incentive for quality over quantity
- Prevents network abuse and low-quality content
- Funds protocol treasury for development

#### Blockchain Verification & Marketplace
- On-chain submission tracking
- Smart contract-based marketplace
- Purchase verification system
- Automated access grant upon payment

#### Purchase System
- Decryption key delivery via SEAL
- Instant access after blockchain confirmation
- Secure key retrieval from key servers
- Browser-based decryption (no server-side processing)
