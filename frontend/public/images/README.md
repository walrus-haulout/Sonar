# SONAR Logo

Please save the SONAR logo image as `sonar-logo.png` in this directory.

The logo should be:
- A PNG file with transparent or dark background
- Minimum 512x512 pixels for best quality
- The stylized walrus with circuit board patterns

Once saved, the logo will appear in:
- Navbar (48x48px)
- Homepage hero section (200x200px)
